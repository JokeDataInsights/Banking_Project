# Banking_Project

## Overview
- This project analyzes a detailed banking dataset to extract insights from customer, transaction, loan, and credit data, with the goal of improving decision-making, customer experience, and financial services.

## Dataset Description
- Customer Information: ID, name, age, gender, address, city, and contact info
- Account Details: Type, balance, account opening date
- Transactions: Dates, types, amounts, and post-transaction balances
- Loans: Types, amounts, interest rates, terms, approval/rejection dates and statuses
- Credit Cards: Card type, credit limits, balances, payment history
- Feedback: Feedback types, resolution status and time
- Anomaly Flags: Possible unusual or suspicious activity markers


##  Objectives
- Analyze customer distribution across cities.  
- Evaluate account balances by type and monthly payment trends.  
- Assess loan disbursement, interest rates, and loan status.  
- Track daily and monthly transaction volumes.  
- Identify top customers by transaction value.  
- Examine credit card usage, balances, and rewards.  
- Detect anomalies in loan patterns.  
- Understand customer feedback to improve service and performance.
